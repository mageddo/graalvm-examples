Running Micronaut on binary executable generated by GraalVM

#### Building binary and running

```bash
$ ./gradlew nativeImage
$ ./build/graal/micronaut
```

#### Testing the API

Setup Postgres

```bash
docker-compose up postgres
```

Testing API

```bash
curl -s localhost:8080/fruits | jq '.'
```
